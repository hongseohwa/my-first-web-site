CSS 스터디 7일차

나를 소개하는 웹사이트

style영역에 h1 선택자를 추가하여 텍스트 편잡

font-size: 45px;를 통해 글자 크기 변경

text-align: center;을 사용해 텍스트를 가운데 정렬

태그에 class="saw"이런 형식으로 속성 지정, .saw는 class가 saw인 모든 태그를 가르키는 선택자

class라고 하는 속성의 값은 여러 개의 값이 올 수 있고, 띄어쓰기로 구분, 하나의 태그에는 여러가지 속성이 들어갈 수 있고, 여러 개의 선택자를 통해 하나의 태그를 공동으로 제어 가능

class에는 순서가 작용함, 태그 > 클래스 > ID 선택자 순으로 강함, 똑같은 선택자라면 맨 마지막에 등장하는 선택자가 우선순위가 높고 ID값은 단 한번만 등장(중복X)

태그 선택자가 id선택자보다 훨씬 더 포괄적, CSS를 만든 사람들은 좀 더 구체적인 것의 우선순위를 포괄적인 것보다 높임
